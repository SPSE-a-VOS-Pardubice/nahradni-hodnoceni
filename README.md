# Náhradní hodnocení

## Instalace
Pro instalaci závislostí je zapotřebí `ext-gd *` a `ext-zip *`
### Windows
???

### Fedora
```bash
sudo dnf install php-gd php-pecl-zip
```

### Závislosti
Projekt používá [Composer](https://getcomposer.org/).
Pro instalaci/aktualizaci závislostí musíte spustit
```bash
composer update
```
