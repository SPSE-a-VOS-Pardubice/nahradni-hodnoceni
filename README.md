### Závislosti
Projekt používá [Composer](https://getcomposer.org/) pro správu závislostí.
Pro instalaci musíte mít Composer v PATH a ve složce projektu spustit
```bash
composer install
```

A pro následné aktualizace
```bash
composer update
```
